Правила игры

Право первого хода предоставляется участнику, осуществляющему игровые действия шашками белого цвета. Выбрать шашку для хода можно с помощью мыши, щелкнув левой клавишей по нужной клетке. Выбранное поле будет выделено цветом. Возможные для хода поля будут обозначены желтым цветом. Сделать ход можно так же с помощью нажатия левой клавишей мыши по нужной позиции. Отменить выбор шашки можно повторным нажатием левой кнопки мыши по ней.

Простая шашка может ходить по диагонали влево и вправо на одну клетку вперед, без возможности осуществления хода назад. Простая шашка может рубить как вперед, так и назад, причем можно рубить несколько вражеских шашек за ход.

Если есть возможность рубить, то это нужно делать обязательно. (Возможность рубить является обязательным игровым ходом.)

Когда шашка достигает противоположной стороны доски, она становится дамкой. На ней появляется изображение короны. Она может ходить, рубить назад, вперед по диагонали на произвольное количество клеток.

Если у одного из игроков после хода соперника не остается ни одной шашки или у него нет возможности ходить (шашки заблокированы), то этот игрок проигрывает. Появляется сообщение о результатах игры.

После каждой партии игроки меняются цветами шашек.

В правой части экрана на специальной панели расположена информация об игре. Там указаны очередность хода, имена игроков и количество оставшихся шашек у каждого игрока.

Также есть возможность вращать доску с шашками по трем пространственным осям. Вращение по оси:

  • Ox осуществляется клавишами-стрелками ВВЕРХ и ВНИЗ.
  • Oy – клавишами-стрелками ВЛЕВО и ВПРАВО.
  • Oz – латинскими буквами "A" и "D" на клавиатуре.

Еще можно приближать или отдалять доску с помощью колеса мыши.
